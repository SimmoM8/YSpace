# YSpace

YSpace is a simple full-stack space travel booking application.

Users can browse futuristic space trips, view destinations, and book journeys to outer space and other planets.

## Features

- View available space travel destinations
- Browse trips and travel details
- Create bookings
- Manage basic user booking data

## Tech Stack

- Frontend: React
- Backend: Node.js / Express
- Database: SQL database

## Project Status

This project is currently in early development.

## Purpose

YSpace was created as a full-stack booking application concept to practise building a complete web app with frontend, backend, and database functionality.
